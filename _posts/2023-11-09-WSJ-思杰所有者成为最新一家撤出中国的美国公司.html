拥有企业软件品牌思杰(Citri)的Cloud Software Group将停止在中国的业务交易，成为最新一家撤出中国市场的美国公司。

Cloud Software Group在《华尔街日报》(The Wall Street Journal)看到的一封周一发给客户和合作伙伴的电邮中表示，已决定于12月3日在中国（包括香港）“停止所有新的商业交易”，其理由是市场成本上升。

该公司表示，将继续为与其签订合同的客户提供维护和支持服务，并对服务中断表示歉意。

Cloud Software Group的一名发言人证实了这封邮件，并表示这一决定影响到其旗下的所有公司。

总部位于佛罗里达州的Cloud Software Group拥有包括思杰和Tibco在内的企业软件供应商。思杰生产办公室软件，允许用户在线访问桌面以及其他云计算功能。该公司向企业提供软件解决方案，并与渠道合作伙伴合作，为终端客户提供服务。

随着中国经济增长放缓，以及对国家安全和数据相关问题的担忧加剧，Cloud Software Group是最新一家撤出或大幅收缩在华业务的美国科技公司。

思杰曾在纳斯达克上市，2022年被Vista Equity Partners和Elliott Management的一个分支机构私有化，后来并入Cloud Software Group。该公司网站显示，在北京、上海、广州和香港设有销售办事处。

思杰曾为中国大型企业提供服务，包括电子商务公司京东(JD.com)和国有汽车制造商上海汽车集团(SAIC)。
